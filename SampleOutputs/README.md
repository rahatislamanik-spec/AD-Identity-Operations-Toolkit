# Sample Outputs

This folder contains sanitized sample report screenshots generated by the AD Identity Operations Toolkit phases.

All outputs are anonymized — no real account names, domain names, or organizational data are present.

> **Note:** Screenshot evidence is captured during live execution against a Windows Server 2022 domain environment. Sample outputs will be added upon next lab execution cycle. Script logic, report structure, and HTML output templates are fully visible in each phase's `.ps1` file.

---

## Expected Report Types

| Phase | Report Type | Description |
|---|---|---|
| Phase 1 | HTML Report | Stale Account Detection — risk-scored findings table |
| Phase 2 | HTML Report | Privileged Account Audit — Tier 0/1 principal inventory |
| Phase 3 | HTML Report | Password Policy Compliance — PSO assessment + violations |
| Phase 4 | HTML Report | Group Membership Audit — circular nesting, empty groups |
| Phase 5 | HTML Report | Service Account Governance — Kerberoastable accounts |
| Phase 6 | HTML Report | Inactive Object Cleanup — remediation pipeline status |
| Phase 7 | HTML Report | Executive Summary — CISO-ready risk dashboard |
