# Sample Outputs

This folder is reserved for future sanitized sample reports and screenshots generated by the AD Identity Operations Toolkit phases.

No generated reports or screenshots are currently retained in this repository.

> **Note:** The script logic, report structure, and HTML output templates are visible in each phase's `.ps1` file. Sanitized sample outputs should be added after the AD lab is rebuilt and the phases are executed again.
